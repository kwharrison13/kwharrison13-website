# SentinelOne

SentinelOne appears across Kyle's 2021 API-security diligence as a named exemplar of the [Endpoint Security](/notes/endpoint-security) category. In his expert reference call, [Nir Rothenberg](/notes/nir-rothenberg) (CISO at [Rapyd](/notes/rapyd)) used it to anchor his "must-have" security-stack framing: *"You have to have endpoint security (CrowdStrike, SentinelOne). You have to have a Firewall (Palo Alto, CheckPoint). You need a CSPM (Orca, Wiz.io). You need to have an API security tool."* In that framing SentinelOne sits beside [CrowdStrike ($CRWD)](/notes/crowdstrike) as one of the two defining vendors of a mature endpoint-security category — the analogy Rothenberg used to argue that API security would become the next inevitable line item in the stack. The wiki references SentinelOne only as this category yardstick, not as a company Kyle diligenced directly.

**Context:** SentinelOne is a publicly traded (NYSE: S) cybersecurity company offering an autonomous, AI-driven Endpoint Detection and Response (EDR) / Endpoint Protection Platform via its Singularity platform — a direct competitor to CrowdStrike in the endpoint-security market.
