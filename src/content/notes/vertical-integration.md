# Vertical Integration

**One-line definition:** *Owning the full care-delivery stack to capture the value of the transaction; [Spring Health](/notes/spring-health)'s core strategy in behavioral health.*

## How sources describe it

- [Spring Health](/notes/spring-health) frames going deeper (medication, substance-abuse, residential care, even high-cost patients) as vertical integration — "why give away that revenue?"; if you're not vertically integrated it's hard to capture the value of the transaction. (September 22nd, 2021 — Adam Chekroud (Spring Health))
