# Bubble

Bubble is a no-code application builder that appears in Kyle's notes as part of the no-code/low-code tool stack discussed in a 2021 customer-reference call about [WhaleSync](/notes/whalesync). In that call, Rob at [Cartooli](/notes/cartooli) — a non-technical builder — described building data products and basic job boards/news feeds "with [Bubble](/notes/bubble), [Webflow](/notes/webflow), etc.," while running into the recurring problem that building an aesthetic front end and wiring up a real data layer were "always two disjointed tasks." Bubble sits in the same toolbox he reached for alongside [Airtable](/notes/airtable), [Webflow](/notes/webflow), and [Zapier](/notes/zapier) before adopting WhaleSync to sync Airtable data into a more attractive front end.

**Context:** Bubble is a visual, no-code platform for building web applications without writing traditional code, letting users design interfaces and define logic and database structures through a drag-and-drop editor.
