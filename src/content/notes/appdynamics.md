# Appdynamics

AppDynamics appears in Kyle's November 8th, 2021 — David O'Neill (APIMetrics) sourcing call as an incumbent application-performance-monitoring tool that, per founder [David O'Neill](/notes/david-o-neill), people had "tried with [Appdynamics](/notes/appdynamics), Dynatrace, etc. but never really worked" for the specific problem of API performance visibility in production. O'Neill's framing positions [APIMetrics](/notes/apimetrics) against AppDynamics and [Postman](/notes/postman) as the easy "bare minimum" options ops teams reach for, arguing that neither delivers the trusted, production-grade API performance data banks actually need.

**Context:** AppDynamics is an application performance management (APM) and observability platform, acquired by Cisco in 2017 for roughly $3.7B.
