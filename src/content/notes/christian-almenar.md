# Christian Almenar

Christian Almenar appears in Kyle's notes as a co-founder of [Monad](/notes/monad), surfaced in a pre-brief from [Max Rimpel](/notes/max-rimpel) that Kyle read on September 17th, 2021 (September 17th, 2021 — Monad Pre-Brief (Max Rimpel)). The brief pairs Almenar with co-founder [Jacolon Walker](/notes/jacolon-walker) (with Amer Deeba as incoming Chief Commercial Officer). Per the notes, Almenar and Walker set out to build "a data platform for security (think 'Fivetran for security')" on the thesis that security is increasingly a data problem — companies battle high fragmentation across security tooling and data, and Monad aims to ETL those siloed sources into a normalized repository with pre-built applications on top.
