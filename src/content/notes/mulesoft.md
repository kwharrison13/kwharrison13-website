# Mulesoft

An enterprise API gateway and integration platform, treated across Kyle's API-security diligence as one of the archetypal incumbent [API Gateways](/notes/api-gateways) — alongside [Kong](/notes/kong), [Apigee](/notes/apigee), AWS API Gateway, Azure API Management, and GCP — that "originally solved management/governance, not security" ([API Gateways](/notes/api-gateways)). In the Salt Security Diligence — 2021 notes it appears as a key integration target: traditional in-house deployed gateways like Mulesoft are important for Salt to discover the APIs that sit behind them, the majority of some customers' APIs are built through Mulesoft, and Salt ships a published Mulesoft integration (mirroring its [Kong](/notes/kong) integration). It is also positioned historically — older middleware names like [Boomi](/notes/boomi) (acquired by Dell) came first, "then you had Mulesoft, etc." Reference calls in the Noname Security - Diligence notes draw on [Vara Imandi](/notes/vara-imandi) of Mulesoft for the gateway-vs-security read, and a [Kong](/notes/kong) engineer flagged Mulesoft as a talent feeder that Kong was poaching from.

Separately, in [The Everything Store](/books/the-everything-store) notes Kyle cites Mulesoft (with Salesforce) as an example of an elite **product-marketing** training ground — part of his "[Talent Vortex](/notes/talent-vortex)" research into the best talent orgs. The company surfaces in Kyle's network too via [Ross Mason](/notes/ross-mason), Mulesoft's founder, who is an LP in [Contrary](/notes/contrary) and the connection through which [Francesco Corea](/notes/francesco-corea) first heard about Contrary.

**Context:** MuleSoft is an integration platform (its Anypoint Platform connects applications, data, and APIs) founded by Ross Mason; it IPO'd in 2017 and was acquired by Salesforce in 2018 for roughly $6.5B.
