# F5

A networking and application-security vendor that surfaces in Kyle's corpus as an adjacent player in the WAF / bot-mitigation space. The single grounded reference comes from the [Signal Sciences](/notes/signal-sciences) page: per [Michael Isbitski](/notes/michael-isbitski) (Salt Security Technical Evangelist), [Signal Sciences](/notes/signal-sciences) "partnered with F5 historically," and F5's products "were more in bot mitigation so they'd catch some API attacks but they didn't have the full lifecycle" (source: [Signal Sciences](/notes/signal-sciences), drawn from the Salt Security Diligence — 2021 reference calls). In that framing F5 sits in the same [Web Application Firewall](/notes/web-application-firewall) / [API Security](/notes/api-security) neighborhood that [Signal Sciences](/notes/signal-sciences) and Salt Security were evaluated against, with bot mitigation as its strength rather than full API-lifecycle coverage.

**Context:** F5 (formerly F5 Networks) is a publicly traded application-delivery and security company, known for its BIG-IP load balancers and a portfolio spanning web application firewalls, bot defense, and API protection. (General background — only the bot-mitigation / Signal Sciences partnership is attested in this wiki.)
