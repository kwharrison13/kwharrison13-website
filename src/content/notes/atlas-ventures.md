# Atlas Ventures

Atlas Ventures appears in Kyle's notes as a named investor in [NeoSec](/notes/neosec), one of the API-security vendors he evaluated during the Salt Security Diligence — 2021 work. In the October 2021 reference call, [Nir Rothenberg](/notes/nir-rothenberg) (CISO at [Rapyd](/notes/rapyd)) described NeoSec ("Neo") as having "raised from Atlas and a lot of VCs in Israel," with a team that had already sold their first startup to [Palo Alto Networks ($PANW)](/notes/palo-alto-networks) and a log-analysis, XDR-style approach that "felt like it was geared towards being acquired by [CrowdStrike ($CRWD)](/notes/crowdstrike)." Atlas is mentioned only as the lead backer behind that Israeli security startup; no further detail about the firm is recorded.

**Context:** "Atlas Ventures" is a name used by more than one venture firm; in an Israeli cybersecurity context it most plausibly refers to an early-stage VC backing security startups. Kyle's source does not disambiguate, so the precise real-world firm is uncertain. This is general background.
