# CrowdStrike

**One-line:** *Publicly traded endpoint-security platform (EDR/MDR); across Kyle's security diligence it functions as the reference analogy and incumbent yardstick — "the CrowdStrike of API security" / "CrowdStrike to Kong's ServiceNow."*

## What we know

- Central company-building analogy in the Salt diligence: [ServiceNow ($NOW)](/notes/servicenow) → CrowdStrike :: [Kong](/notes/kong) → [Salt Security](/notes/salt-security) — the management layer spawns a dedicated security company on top. (Salt Security Diligence — 2021, per [Atanu Dasgupta](/notes/atanu-dasgupta))
- Anchors the "must-have" security-stack framing: "you have to have endpoint security (CrowdStrike, SentinelOne)... a Firewall (Palo Alto, CheckPoint)... a CSPM (Orca, Wiz.io)... [and] an API security tool." (Salt Security Diligence — 2021, per [Nir Rothenberg](/notes/nir-rothenberg))
- Doesn't extend into Application Security because "it's very different than endpoint security" — cited as the limit on gateway/EDR players expanding into API security. (Salt Security Diligence — 2021, per [Atanu Dasgupta](/notes/atanu-dasgupta))
- [NeoSec](/notes/neosec) was described as feeling "geared towards being acquired by CrowdStrike, like 'CrowdStrike for API Security.'" (Salt Security Diligence — 2021, per [Nir Rothenberg](/notes/nir-rothenberg))
- Used as the maturity benchmark for where API security is headed — API security "feels like Endpoint Security 5-6 years ago," and CrowdStrike's strength is packaging indicators into an actionable report ("click here if you want us to stop X") vs. legacy tools that dump raw signal. (Noname Security - Diligence, per [Ian Buxton](/notes/ian-buxton) of [Vail Resorts](/notes/vail-resorts))
- Acts as a competitive gatekeeper in the endpoint market: the CrowdStrike team reportedly shut down a [CapitalG](/notes/capitalg) term sheet for rival [Uptycs](/notes/uptycs), which Comcast evaluated side-by-side against CrowdStrike. (September 22nd, 2021 — Ganesh Pai (Uptycs))
