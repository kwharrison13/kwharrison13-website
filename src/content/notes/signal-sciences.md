# Signal Sciences

**One-line:** *WAF (Web Application Firewall) company acquired by [Fastly](/notes/fastly); evaluated during Salt Security diligence as an API-security option but judged a strong WAF, not a full API security platform.*

## What we know

- Acquired by [Fastly](/notes/fastly), which was competing with [Cloudflare](/notes/cloudflare) around application firewalls; the acquisition gives Fastly much broader outreach to enterprise customers. ([Prabhath Karanth](/notes/prabhath-karanth), Salt Security Diligence — 2021)
- "Pretty mature business without a ton of fast-moving innovation." ([Prabhath Karanth](/notes/prabhath-karanth), Salt Security Diligence — 2021)
- "Great WAF. Really close to understanding the implications of their business but its a WAF." It was evaluated alongside [Salt Security](/notes/salt-security) ~3 years earlier; good (not great) at API stuff, so not worth switching just for a slightly better WAF. ([Nir Rothenberg](/notes/nir-rothenberg), Salt Security Diligence — 2021)
- Agent-based model — you must install an agent on every server, which creates a lot of operational overhead and risks missing APIs without full agent coverage. ([Nir Rothenberg](/notes/nir-rothenberg), Salt Security Diligence — 2021)
- A reviewer who evaluated it "more from a business point of view" flagged it as recently acquired. ([Atanu Dasgupta](/notes/atanu-dasgupta), Salt Security Diligence — 2021)
- Partnered with F5 historically; "they were more in bot mitigation so they'd catch some API attacks but they didn't have the full lifecycle." ([Michael Isbitski](/notes/michael-isbitski), Salt Security Technical Evangelist, Salt Security Diligence — 2021)
