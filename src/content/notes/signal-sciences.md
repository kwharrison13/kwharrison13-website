# Signal Sciences

**One-line definition:** *WAF (Web Application Firewall) company acquired by [Fastly](/notes/fastly); evaluated as an API security tool but ultimately a WAF with strong bot mitigation, not a full API security platform.*

## How sources describe it

- Acquired by [Fastly](/notes/fastly) (which was competing with [Cloudflare](/notes/cloudflare) around application firewalls); gives Fastly broader enterprise reach. ([Prabhath Karanth](/notes/prabhath-karanth), Salt Security Diligence — 2021)
- "Great WAF. Really close to understanding the implications of their business but its a WAF." ([Nir Rothenberg](/notes/nir-rothenberg), Salt Security Diligence — 2021)
- Agent-based model (must install on every server) — creates significant operational overhead; will miss APIs without full agent coverage.
- "Pretty mature business without a ton of fast-moving innovation." ([Prabhath Karanth](/notes/prabhath-karanth), Salt Security Diligence — 2021)
- Partnered with F5 historically; strong in bot mitigation but lacked full API lifecycle security. ([Michael Isbitski](/notes/michael-isbitski), Salt Security Diligence — 2021)

## Where it shows up

- Salt Security Diligence — 2021 — evaluated as a competitor by multiple reference call subjects.
