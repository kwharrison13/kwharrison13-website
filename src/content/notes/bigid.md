# BigID

Data privacy / data security company, raised by Kyle in a note on [Todd McKinnon — Creating and Defining a New Market Category](/notes/todd-mckinnon-creating-and-defining-a-new-market-category) alongside [OneTrust](/notes/onetrust) as a category that [Okta ($OKTA)](/notes/okta) might compete with or acquire into.

**Context:** BigID is a data intelligence platform for data discovery, privacy, security, and governance — helping enterprises find and classify sensitive data across their environments. It is frequently grouped with OneTrust in the data privacy / governance category.
