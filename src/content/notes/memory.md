# Memory

Memory recurs across Kyle's learning, development, and gospel reading as the throughline that connects encoding, understanding, and durable knowledge. [The Boy Who Was Raised as a Dog](/books/the-boy-who-was-raised-as-a-dog) gives the developmental-neuroscience frame: memory is "the capacity to carry forward in time some element of an experience," the brain is "an historical organ" storing template memories that guide behavior, and to create an effective memory "experience has to be patterned and repetitive." That same book flags the shadow side — because of how memory works, ruminating on past negative events can recolor old, ambiguous memories darker over time until they become "a trauma that never actually occurred."

In the knowledge-work strand, memory is about retrieval, not storage. [How To Take Smart Notes](/books/how-to-take-smart-notes) argues that "memory is about retrieval strength, not storage" — attach information to a framework (Munger's [Latticework](/notes/latticework) of mental models) so the right cues trigger it — and that the best-researched learning method is [elaboration](/notes/elaborate): rephrasing an idea in your own words and connecting it to prior knowledge so it forms durable long-term memory. [Why Books Don't Work](/notes/why-books-don-t-work) pairs memory with [Spaced Repetition](/notes/spaced-repetition) as the two cognitive-science ideas [Andy Matuschak](/notes/andy-matuschak) weaves into a prototype medium ([Quantum Country](/notes/quantum-country)), on the premise that a thing's prerequisites must be "not just familiar but fluent, encoded in long-term memory." [Joseph Smith Lecture 2 — Joseph's Personality and Character](/notes/joseph-smith-lecture-2-joseph-s-personality-and-character) treats memory as one of Joseph Smith's "four casts of mind," illustrated by his dictating D&C 132 straight through and claiming he could rewrite it from memory.

**Context:** In cognitive science, long-term memory is generally distinguished from a limited short-term/working memory (famously "seven plus or minus two" items), and durable retention is strengthened by elaboration, retrieval practice, and spaced review timed against the forgetting curve.
