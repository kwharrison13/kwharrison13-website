# Palo Alto Networks

**One-line:** *Publicly traded cybersecurity platform (firewall, cloud security / CSPM); appears in Kyle's API-security diligence as an incumbent comparison and acquisition reference point rather than a diligence subject.*

## What we know

- Positioned in the firewall/CheckPoint tier of the "must-have" security stack — "you need a CSPM (Orca, Wiz.io). You need to have an API security tool." (Salt Security Diligence — 2021, per [Nir Rothenberg](/notes/nir-rothenberg))
- Acquired [Bridgecrew](/notes/bridgecrew) (cloud security policy); this was cited as the comparable prior acquisition when [NeoSec](/notes/neosec) was described as feeling "geared towards being acquired by CrowdStrike." (Salt Security Diligence — 2021, per [Nir Rothenberg](/notes/nir-rothenberg))
- Cited as a slower innovator — "if you look at Cisco, Palo Alto Networks, etc. their propensity to innovate is much slower" — used to explain why [TripActions](/notes/tripactions) chose startup [Salt Security](/notes/salt-security) over incumbents. (Salt Security Diligence — 2021, per [Prabhath Karanth](/notes/prabhath-karanth))
- Named among the big firewall/WAF (layer-7 inspection) incumbents alongside [Imperva](/notes/imperva) in the Noname dossier's Kong call. (Salt Security Diligence — 2021)
