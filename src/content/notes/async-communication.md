# Async Communication

A shorthand for the same idea covered in full on [Asynchronous Communication](/notes/asynchronous-communication) — the working mode where collaboration happens through written text rather than in real time. On the [Open Source Knowledge](/notes/open-source-knowledge) page this thread is drawn from [Remote — Office Not Required](/books/remote-office-not-required): the prompt "what other types of work can you open source?" reframes async, written communication as a way to share expertise openly, with [Writing](/notes/writing) as the core skill of distributed knowledge work.

See [Asynchronous Communication](/notes/asynchronous-communication) for the primary note.
