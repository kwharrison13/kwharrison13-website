# Transmit Security

Transmit Security shows up in Kyle's 2021 security diligence as a **passwordless / customer-authentication player**. In the IAM diligence, it sits in the Passwordless investment-thesis bucket alongside DoubleOctopus, and an expert (Aaron Perry) characterizes it as "more similar to Auth0 than [Okta](/notes/okta)" — a drag-and-drop authentication engine embedded in the IDE, white-label and API-driven, that can pair with Auth0 or Okta for initial authentication. The catch: it's "a very technical sale," often into KYC/fraud, and "not something you can just download and try," which makes it harder to sell than Auth0.

The connection to Kyle's other dossier runs through the investor: in the Noname Security diligence, [Gili Raanan](/notes/gili-raanan) of [Sequoia](/notes/sequoia)/Cyberstarts lists Transmit Security as a portfolio company (alongside Armis, Wiz, and Noname) while pitching Noname — making it a Cyberstarts / Gili Raanan comparable.

**Context:** Transmit Security is a Boston/Tel Aviv-based identity-security company building passwordless authentication and customer identity (CIAM) / fraud-prevention products; it raised a large Series A in 2021.
