# Jacolon Walker

Co-founder of [Monad](/notes/monad), alongside Christian Almenar, with Amer Deeba as incoming Chief Commercial Officer. Per the [Max Rimpel](/notes/max-rimpel) pre-brief Kyle read on September 17th, 2021, Almenar and Walker built Monad on the thesis that security is increasingly a data problem — companies battle high fragmentation across security tooling and data — and set out to build a "Fivetran for security" that ETLs siloed security data sources into a normalized repository, with pre-built applications (vulnerability management, team performance, tooling ROI, benchmarks) layered on top.
