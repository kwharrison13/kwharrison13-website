# Daily Notes

"Daily Notes" is Roam Research's signature feature — a date-stamped page that greets the user each day, giving them an opinionated place to start writing. In [The Notetaking Cold War](/notes/the-notetaking-cold-war), it is framed as Conor White-Sullivan's way of sneaking a well-defined process into Roam's feature set: without it, Roam's blank page would intimidate most users and they'd never build anything.

## Where this appears
