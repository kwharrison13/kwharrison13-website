# MongoDB

MongoDB shows up across two of Kyle's diligence dossiers in distinct roles. In Pave Diligence — 2021, MongoDB is a customer reference: [Alex Koykova](/notes/alex-koykova) at MongoDB is cited as a [ChartHop](/notes/charthop) user, and her testimonial — that ChartHop "helped us centralize over a hundred headcount planning spreadsheets into a single platform" — is used to characterize the competitive landscape Pave was operating in.

In Starburst - Diligence, MongoDB appears twice as a market analogy and a data source. The notes use MongoDB as a benchmark for open-source data-infrastructure momentum — observing that Starburst's "tech is 'just ok' — but MongoDB tech is 'just ok'; the question is if you have the momentum of the market" — and list MongoDB among the data sources (PostgreSQL, MySQL, Hive) that Starburst's SQL query layer federates across.

**Context:** MongoDB is a publicly traded ($MDB) database company built around its document-oriented NoSQL database and the Atlas managed cloud service.

## Where this appears

- Pave Diligence — 2021 — Alex Koykova at MongoDB cited as a ChartHop reference customer in the competitive landscape.
- Starburst - Diligence — used as an analogy for open-source data-infra market momentum, and listed as a queryable data source under Starburst's SQL federation layer.
