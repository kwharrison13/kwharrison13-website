# Leadership Compass

[KuppingerCole](/notes/kuppingercole)'s signature analyst-report format — a comparative market evaluation of vendors in a given category. In Kyle's notes it surfaces through the API security space: [Alexei Balaganski](/notes/alexei-balaganski), the KuppingerCole analyst who has tracked API security since 2014, authored the *API Management and Security Leadership Compass* (alongside an *API Management & Security Buyer's Compass*). The report appears in the Salt Security Diligence — 2021 dossier flagged "To Read" as a primary-source read on the API security market and its ~31 meaningful vendors.

**Context:** The Leadership Compass is KuppingerCole's branded analyst format for ranking vendors within a market segment on criteria like product, innovation, and market presence — roughly analogous to Gartner's Magic Quadrant or Forrester's Wave.
