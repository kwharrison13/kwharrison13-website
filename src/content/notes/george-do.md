# George Do

**One-line:** *SVP & CISO at [Gojek](/notes/gojek) (Southeast Asia super-app) and concurrently CISO at Equinix; ranks API security above endpoint security for mobile-first API-native companies.*

## What we know

- SVP & CISO at [Gojek](/notes/gojek), a Southeast Asian super-app (payments, food delivery, transportation, logistics) with 100M users. Group CISO responsible for all units, cybersecurity, and data privacy across 4 countries.
- Concurrently CISO at Equinix; implemented global security platforms across 200+ locations.
- At time of interview (October 2021), evaluating [Salt Security](/notes/salt-security) deeply and planning to invest.
- Priority ranking: API security is #2 (behind only data privacy/security), above endpoint or network security — driven by Gojek's mobile-first, 100% API-based architecture.
- TAM view: "$10–20B addressable market"; beyond that "the big guys (Microsoft, Amazon, Google) will start to wake up."
- Uses [Kong](/notes/kong) as API Gateway; believes Kong could theoretically build into API security but would take time.
- Contact: george.do@gojek.com (source: Salt Security Diligence — 2021).
