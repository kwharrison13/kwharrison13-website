# Cloudflare

**One-line definition:** *CDN, DNS, and edge security platform; identified as a potential API security threat / acquirer given existing edge infrastructure and enterprise reach.*

## Where it shows up

- Salt Security Diligence — 2021 — mentioned as a potential entrant into API security (competing with [Fastly](/notes/fastly) around application firewalls; edge players could logically expand to API security). [David Mark](/notes/david-mark) (Appsflyer) suggested API security companies could integrate with Cloudflare for active remediation.
