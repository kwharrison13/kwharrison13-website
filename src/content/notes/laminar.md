# Laminar

**One-line definition:** *Data security posture management (DSPM) platform; complements API security tools — customers buying Laminar were also buying API security products like [Salt Security](/notes/salt-security), seeing them as coexisting.*

## How sources describe it

- "Many customers that are evaluating Laminar have also adopted API Security products; they see them as complementary and are willing to pay for both." (Salt Security Diligence — 2021)
- Raises the strategic question: "Why would you want to protect individual components of your infrastructure when you can just protect your data at the point of creation?" ([Alexei Balaganski](/notes/alexei-balaganski), Salt Security Diligence — 2021)
- Laminar's strategy: coexist with API security companies for now, make them redundant over time as Laminar can support API security functionality without an additional product.

## Where it shows up

- Salt Security Diligence — 2021 — referenced as the "protect all data" alternative angle to API-component security.

## Related concepts

- [API Security](/notes/api-security) — the adjacent/overlapping category Laminar intends to absorb
