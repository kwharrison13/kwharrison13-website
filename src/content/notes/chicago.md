# Chicago

**One-line definition:** *The City of Chicago — referenced for its mature municipal data operation under deputy IT director [Tom Schenk](/notes/tom-schenk).*

## How sources describe it

- Runs a full data operation across database management, BI, data publication, and advanced analytics, with an expanded Open Data Portal and tools like Open Grid / Windy-Grid. (June 1st, 2018 — Tom Schenk (City of Chicago))
- Co-wrote legislation legalizing ride shares that required per-trip data (driver, vehicle), now used to understand mobility — one of the few cities (with New York) requiring such data access.
- Avoids procurement friction via resellers like CDW/CDWG; pursues practical service-delivery analytics (food-inspector routing, lead-paint prediction) and identifies Chicago-based software companies whose products might serve cities.
