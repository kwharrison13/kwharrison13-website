# Elastic Beam

Elastic Beam is the [API Security](/notes/api-security) company that [Ping Identity](/notes/ping-identity) acquired, cited in the 2021 [Salt Security](/notes/salt-security) diligence as a cautionary case for how identity/gateway vendors under-invest in runtime API security. Per [Michael Isbitski](/notes/michael-isbitski) (Salt's Technical Evangelist, ex-[Gartner](/notes/gartner)), once Ping absorbed Elastic Beam, API security "wasn't a core focus" — because, in his framing, "API Gateway buyers aren't the security teams; they're not focused on runtime security that deep." The point supports the diligence thesis that incumbents like Ping and [Apigee](/notes/apigee) structurally fail to productize deep API-runtime security, leaving room for a dedicated player like Salt Security.

**Context:** Elastic Beam was an API cybersecurity startup using AI/behavioral analysis to protect APIs; Ping Identity acquired it in 2018 and built it into PingIntelligence for APIs.
