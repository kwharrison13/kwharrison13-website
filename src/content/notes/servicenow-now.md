# ServiceNow ($NOW)

**One-line definition:** *Enterprise workflow / IT service management platform; used as the "$15B endpoint management analogy" in the Salt Security thesis — the management layer that spawned CrowdStrike.*

## How sources describe it

- Central analogy in the Salt Security diligence: ServiceNow → [CrowdStrike ($CRWD)](/notes/crowdstrike-crwd) :: [Kong](/notes/kong) → [Salt Security](/notes/salt-security) — the API management layer (Kong) spawns a dedicated security company (Salt). (Salt Security Diligence — 2021)
- "ServiceNow don't really do security. CrowdStrike is all over the place (EDR, MDR, etc.)." — management platform and security platform are different enough to justify separate companies. ([Atanu Dasgupta](/notes/atanu-dasgupta), Salt Security Diligence — 2021)
- Endpoints ($15B market) = ServiceNow → CrowdStrike; API Management ($22B market) = Kong → Salt Security. (Salt Security Diligence — 2021)
