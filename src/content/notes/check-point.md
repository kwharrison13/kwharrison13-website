# Check Point

Check Point appears in Kyle's security diligence as the firewall incumbent in the "must-have" security-stack framing. Per the Salt Security diligence, the stack runs: endpoint security ([CrowdStrike](/notes/crowdstrike), SentinelOne), a firewall (Palo Alto, CheckPoint), a CSPM ([Orca Security](/notes/orca-security), [Wiz.io](/notes/wiz-io)), and an API security tool — with Check Point named alongside [Palo Alto Networks](/notes/palo-alto-networks) as the firewall layer. It is invoked as a category anchor and incumbent reference point rather than as a diligence subject in its own right.

**Context:** Check Point Software Technologies is an Israeli cybersecurity company founded in 1993, one of the original network-firewall pioneers (its FireWall-1 product helped commercialize stateful inspection). It remains a publicly traded incumbent across network, cloud, and endpoint security.

## Where this appears

- [CrowdStrike](/notes/crowdstrike) — listed in the "must-have" stack as the firewall layer (Palo Alto, CheckPoint) beneath endpoint security.
- [Palo Alto Networks](/notes/palo-alto-networks) — paired with PANW as the firewall/CheckPoint tier of that same security stack.
