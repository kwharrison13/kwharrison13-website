# Boomi

Boomi appears in Kyle's API-security diligence as one of the **incumbent integration / API-gateway players**, almost always named in the same breath as [Mulesoft](/notes/mulesoft) and [Kong](/notes/kong). In the Noname Security - Diligence notes, [Vara Imandi](/notes/vara-imandi) of [Mulesoft](/notes/mulesoft) lists "[Kong](/notes/kong) or even Boomi" as other players "leading with an API Gateway narrative." In the Salt Security Diligence — 2021 notes, [Prabhakar Kasu](/notes/prabhakar-kasu) (AON) frames Boomi as part of the historical arc of the gateway market — "older names like Boomi that got acquired by Dell maybe a decade ago, then you had [Mulesoft](/notes/mulesoft), etc." — and notes the core problem these gateways originally solved was management/governance, not security. So in Kyle's framing Boomi is a legacy API-management / integration incumbent that predates the modern API-security category, not an active deal.

**Context:** Boomi is an integration platform-as-a-service (iPaaS) / API management vendor, founded in 2000 and acquired by Dell in 2010 (operating as Dell Boomi), then spun out to private equity (Francisco Partners and TPG) in 2021.
