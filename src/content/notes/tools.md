# Tools

Recurring theme in *How to Take Smart Notes*: "even the best tools won't make much of a difference if they are used in isolation... only if they are embedded in a well-conceived working process can the tools play out their strengths." The book specifies just four tools needed for the slip-box workflow — something to capture fleeting ideas, a reference manager, the slip-box itself, and an editor — arguing that more tools add friction rather than value, and that a tool's usefulness depends on the routine it's embedded in, not the tool alone.
