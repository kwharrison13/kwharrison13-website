# 42 Crunch

42 Crunch surfaces in Kyle's diligence call with David O'Neill of [APIMetrics](/notes/apimetrics) as a potential partner on the API-security side. O'Neill described APIMetrics "starting to talk to folks at 42 Crunch to offer security monitoring as well" — reviewing API specs and looking for holes in design — as a complement to APIMetrics' API performance monitoring, in the context of becoming a "nexus of verifying compliance and governance in production within a secure environment" for banking and fintech customers.

**Context:** 42 Crunch is an API security company known for tools that audit OpenAPI/Swagger specifications, detect vulnerabilities, and enforce security at runtime via micro-API firewalls.
