# Aqua Security

Aqua Security shows up in Kyle's API-security diligence as a reference point for **posture management** — the security capability that scans a server (or cloud workload) and surfaces the problems on it, e.g. what you need to patch. In the Noname Security - Diligence notes, [Nir Rothenberg](/notes/nir-rothenberg) (CISO at [Rapyd](/notes/rapyd)) defines a good API-security company as one that does both baseline traffic analysis (table stakes) *and* posture management, and names "[Palo Alto Networks ($PANW)](/notes/palo-alto-networks), Aqua Security, etc." as the players who "take a server and show you any problems." Aqua is cited there as illustrative of posture-scanning security rather than as a deal under evaluation.

It also appears on the [Palo Alto Networks ($PANW)](/notes/palo-alto-networks) page in the cloud-security-posture-management (CSPM) peer set, grouped with [Orca Security](/notes/orca-security) and [Wiz.io](/notes/wiz-io) as comparable players in that category.

**Context:** Aqua Security is an Israeli-American cloud-native security vendor known for a cloud-native application protection platform (CNAPP) spanning container/Kubernetes security, vulnerability scanning, and cloud security posture management.
