# Max Rimpel

Investor who authored and shared the [Monad](/notes/monad) pre-brief Kyle read on September 17th, 2021. The brief pitched Monad as "Fivetran for security" — a data platform built by co-founders [Christian Almenar](/notes/christian-almenar) and [Jacolon Walker](/notes/jacolon-walker) (with [Amer Deeba](/notes/amer-deeba) as incoming Chief Commercial Officer) to ETL fragmented, siloed security data into a normalized repository, then run pre-built applications (vulnerability management, team performance, tooling ROI, benchmarks) on top. Rimpel's brief framed the thesis that "security is increasingly a data problem" and positioned Monad as additive to existing tooling, with [SIEM](/notes/siem) players like [Splunk](/notes/splunk) and [Cribl](/notes/cribl) treated as only mildly competitive.

**Context:** Max Rimpel is a general partner at Andreessen Horowitz (a16z); the pre-brief format and "Fivetran for security" framing are consistent with a venture investor's diligence memo on Monad.
