# Dynatrace

Dynatrace comes up in Kyle's November 2021 call with [David O'Neill](/notes/david-o-neill) of APIMetrics as a point of competitive contrast. Discussing the problem of limited visibility into API performance analytics, O'Neill noted that incumbents like [Appdynamics](/notes/appdynamics) and Dynatrace had tried to address the space "but never really worked" for the kind of cross-party, bank-grade API performance disputes APIMetrics targets. The mention frames Dynatrace as an established application-performance-monitoring player whose general-purpose approach left a gap APIMetrics aimed to fill.

**Context:** Dynatrace is a publicly traded (NYSE: DT) software-intelligence and observability company providing application performance monitoring, infrastructure monitoring, and AI-driven analytics for large enterprises.
