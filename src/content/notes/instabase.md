# Instabase

Instabase appears in Kyle's Pave Diligence — 2021 notes as one of the named [Index](/notes/index-ventures) portfolio companies contributing to Pave's give-to-get compensation data set. The diligence describes how Pave built "a core and proprietary data set for compensation data... operated through a give-to-get model that 1,500 companies are now contributing to (including 40+ Index portfolio companies like Instabase, [Anrok](/notes/anrok), [Discord](/notes/discord), [Newfront Insurance](/notes/newfront-insurance), [Scale AI](/notes/scale-ai), [Boulevard](/notes/boulevard), and [Applyboard](/notes/applyboard))." Instabase is cited as evidence of the network density underpinning Pave's data moat, not analyzed on its own merits.

**Context:** Instabase is an enterprise software company building a platform for automating document- and content-centric business processes (deep learning–based data extraction), backed by investors including Index Ventures.
