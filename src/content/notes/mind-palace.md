# Mind Palace

A memory technique that attaches vivid mental images to a sequence of items so they can be recalled in order — the constructive use of "extra-vivid images" Munger flags in [Charlie Munger — The Psychology of Human Misjudgment](/notes/charlie-munger-the-psychology-of-human-misjudgment) as a device for improving one's own memory.

**Context:** The Mind Palace (or memory palace / method of loci) is an ancient mnemonic technique in which information is encoded as striking images placed along a familiar spatial route; recalling the route walks the mind back through the stored images. It traces to classical rhetoric and remains a staple of competitive memory athletes.

## Where this appears

- [Charlie Munger — The Psychology of Human Misjudgment](/notes/charlie-munger-the-psychology-of-human-misjudgment) — Munger notes that the special power of extra-vivid images can be used constructively to improve one's own memory by attaching images to items one doesn't want to forget.
