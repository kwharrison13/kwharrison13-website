# Orca Security

Orca Security is the cloud-security-posture-management (CSPM) vendor that recurs across Kyle's 2021 cybersecurity diligence as the canonical CSPM comparable, almost always paired with [Wiz.io](/notes/wiz-io). It anchors the "must-have security stack" framing voiced by [Nir Rothenberg](/notes/nir-rothenberg) (CISO of [Rapyd](/notes/rapyd)) in the Noname Security - Diligence: *"You have to have endpoint security ([CrowdStrike ($CRWD)](/notes/crowdstrike), SentinelOne). You have to have a Firewall ([Palo Alto Networks ($PANW)](/notes/palo-alto-networks), CheckPoint). You need a CSPM (Orca, Wiz.io). You need to have an API security tool."* Rothenberg is himself a customer of Orca (alongside CrowdStrike and formerly Palo Alto), making Orca the named CSPM layer in his own real-world stack.

In the September 17th, 2021 — Wiz.io Pre-Brief (Shardul Shah), Orca surfaces as Wiz.io's principal direct competitor — Wiz's main rivals are framed as [Palo Alto Networks ($PANW)](/notes/palo-alto-networks) (which "fumbled its first two acquisitions" in the space) and Orca Security, with well-funded startups (Apiiro and others) expected to pile into the category. Orca thus sits at the intersection of two of Kyle's security threads: the cloud-posture layer that buyers treat as table stakes, and the competitive set the Wiz investment thesis had to clear.

**Context:** Orca Security is an Israeli-founded cloud security company (founded 2019) known for pioneering agentless, "SideScanning" cloud security posture management across AWS, Azure, and GCP.
