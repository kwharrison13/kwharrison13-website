# Innovator's University

Kyle's shorthand, used in a note on [Charlie Munger — A Lesson on Elementary Worldly Wisdom](/notes/charlie-munger-a-lesson-on-elementary-worldly-wisdom), for the idea that institutions don't all need to imitate the top tier: "not everyone should be Harvard, and not everyone should be Sequoia." It frames a differentiated, innovator-oriented model of education (or venture institutions) rather than every player chasing the same elite template.
