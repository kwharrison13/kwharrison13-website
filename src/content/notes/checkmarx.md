# Checkmarx

**One-line definition:** *Application security testing platform (SAST/DAST); the pre-existing tool [AON](/notes/aon) was using before adopting [Salt Security](/notes/salt-security) — does application testing but not API-specific security.*

## Where it shows up

- Salt Security Diligence — 2021 — [Prabhakar Kasu](/notes/prabhakar-kasu) (AON) noted Checkmarx as their existing application testing tool; "not API" — moving to Salt Security to fill that gap.
